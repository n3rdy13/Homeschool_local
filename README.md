# Homeschool Companion & Planner

This is a complete, structured homeschool planning and lesson generation system. It generates personalized learning materials using the Gemini API, tracks student attendance and grades, and provides adaptive learning recommendations.

## Run Locally

### Prerequisites

You need **Node.js** (v18 or higher) installed on your system.

### Setup and Running Instructions

1. **Extract/Download the Codebase:**
   Make sure all files are in a folder on your computer.

2. **Create a `.env` file:**
   Create a `.env` file in the root directory (where `package.json` is located) and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_actual_gemini_api_key_here
   ```

3. **Install Dependencies:**
   Open your terminal in the root directory and run:
   ```bash
   npm install
   ```

4. **Run the Application:**
   Start the development server with:
   ```bash
   npm run dev
   ```

5. **Access the Web App:**
   Open [http://localhost:3000](http://localhost:3000) in your web browser.
